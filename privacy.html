Privacy Policy

This application ("fbbot") is a personal content-publishing tool used by its operator to publish videos to Facebook Pages that the operator owns or manages. It is not offered as a public service to third parties.

Information We Collect

The application does not collect, store, or process any personal data from end users or visitors. The only data handled by the application is:

  • Facebook Page access tokens, provided by the operator, used solely to publish content via the official Meta Graph API.
  • YouTube video URLs that the operator chooses to re-publish.

How We Use Information

Access tokens are stored locally on the operator's private server and are used exclusively for publishing videos to Facebook Pages controlled by the operator. No data is used for analytics, advertising, profiling, or any other purpose.

Data Sharing

No data is shared with any third party. The application communicates only with the official Meta Graph API and YouTube to perform its intended functions.

Data Retention

Downloaded videos are stored only temporarily on the server during the publishing process and are deleted afterwards. Access tokens are retained only as long as the operator continues to use the application.

Security

The application runs on a private server accessible only to the operator. Access is protected by standard authentication mechanisms.

Changes to This Policy

This privacy policy may be updated from time to time. The latest version will always be available at this URL.

Contact

For questions regarding this privacy policy, please contact the application operator directly.
